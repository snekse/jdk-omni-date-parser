# Step 4 — Update README

Find the **Supported Formats** section in `README.md` and add the new format or keyword.
Follow the existing style (bullet points or table rows with a brief example string in
backticks).

If the feature has a scope constraint, note it inline. Examples:

- `noon` / `midnight` — only valid when the hour component is exactly `12`
- A new separator style — note which format families it applies to

Do not add lengthy prose. One line per feature is usually enough.
